不可改变自身的 Array API

- Array.prototype.map

::: tip Author
回答者: [Kiera569](https://github.com/Kiera569)
:::

不改变原数组的方法：concat/join/reduce/map/forEach/filter/slice/findIndex

改变原数组的方法：push/unshift/pop/shift/sort/splice/reverse
