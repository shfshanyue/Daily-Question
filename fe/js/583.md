# 为何 0.1+0.2 不等于 0.3，应如何做相等比较

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

0.1，0.2 表示为二进制会有精度的损失，比较时可引入一个很小的数值 `Number.EPSILON` 容忍误差，其值为 2^-52。

```js
function equal(a, b) {
  return Math.abs(a - b) < Number.EPSILON;
}
```
