# 关于 this 与包装对象，以下输出多少

::: tip 更多描述

```js
function foo() {
  console.log(this);
}

foo.call(3);
```

:::

如果处于非严格模式下，要绑定的`this`指定为`null`或`undefined`时会自动替换为全局对象，原始值则会被包装

严格模式：

```js
"use strict";

function test() {
  console.log(this);
}

test.call(2);
// 2
```

非严格模式

```js
function test() {
  console.log(this);
}
test.call(2);
// Number {2}
```
