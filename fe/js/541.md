# 解构赋值以下对象，他们的值是多少

::: tip 更多描述

---

```js
const { a: aa, b } = { a: 3, b: 4 };
```

分别打印 `a`、`aa`、`b`，他们的值是多少

---

```js
function move({ x = 0, y = 0 } = {}) {
  return [x, y];
}

function move({ x, y } = { x: 0, y: 0 }) {
  return [x, y];
}
```

以下几个表达式返回的结果是什么

```js
move({ x: 3, y: 8 });
move({ x: 3 });
move({});
move();
```
