# JS 如何翻转一个字符串

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 470](https://github.com/shfshanyue/Daily-Question/issues/470)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

```js
const reverse = s =>
  s
    .split("")
    .reverse()
    .join("");
```

```js
const reverse = s => [...s].reduceRight((acc, cur) => acc + cur);
```
