# textarea 如何禁止拉伸

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 484](https://github.com/shfshanyue/Daily-Question/issues/484)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

使用 CSS 样式可以避免拉伸

```css
textarea {
  resize: none;
}
```
