# react hooks 中如何模拟 componentDidMount

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 143](https://github.com/shfshanyue/Daily-Question/issues/143)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

在 `useEffect`，把第二个参数即依赖的状态，设置为 `[]`

```js
useEffect(callback, []);
```
