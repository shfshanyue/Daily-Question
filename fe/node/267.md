# Node 应用中如何查看 gc 的日志

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 267](https://github.com/shfshanyue/Daily-Question/issues/267)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

通过开启参数 `--trace-gc` 与 `--trace-gc-verbose`

$ node --trace-gc index.js
