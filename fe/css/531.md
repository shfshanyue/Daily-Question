# CSS 如何设置多行超出显示省略号

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 531](https://github.com/shfshanyue/Daily-Question/issues/531)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

使用 `-webkit-line-clamp` 来设置多行超出显示省略号

```css
overflow: hidden;
display: -webkit-box;
-webkit-box-orient: vertical;
-webkit-line-clamp: 2;
```
