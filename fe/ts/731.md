和 returnType 有点关联，做返回值推断的

::: tip Author
回答者: [canvascat](https://github.com/canvascat)
:::

`infer` 表示在 `extends` 条件语句中待推断的类型变量。

例子：

```ts
/**
 * Obtain the parameters of a function type in a tuple
 */
type Parameters<T extends (...args: any) => any> = T extends (
  ...args: infer P
) => any
  ? P
  : never;
```
