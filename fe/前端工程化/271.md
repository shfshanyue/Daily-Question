# 什么是 XSS 攻击，如何避免

CSS (Cross Site Scripting)，跨站脚本攻击。可使用以下脚本在指定网站上进行攻击

```html
<script>
  alert("XSS");
</script>

<img src="https://devtool.tech/notfound.png" onerror="alert('XSS')" />
```
