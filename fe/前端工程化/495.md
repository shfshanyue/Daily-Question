# 什么是前端工程化

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 495](https://github.com/shfshanyue/Daily-Question/issues/495)
:::

::: tip Author
回答者: [haotie1990](https://github.com/haotie1990)

前端工程化的主要目标就是解放生产力、提高生产效率。通过制定一系列的规范，借助工具和框架解决前端开发以及前后端协作过程中的痛点和难度问题。
