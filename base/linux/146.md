# 如何判断端口是否可达

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 146](https://github.com/shfshanyue/Daily-Question/issues/146)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

使用 `nc`，`-z` 指测试接口连通性

```bash
nc -vz localhost 443
```
