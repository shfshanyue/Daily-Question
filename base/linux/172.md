# 在服务器内如何得知自己的公网 IP

::: tip Issue
欢迎在 Gtihub Issue 中回答此问题: [Issue 172](https://github.com/shfshanyue/Daily-Question/issues/172)
:::

::: tip Author
回答者: [shfshanyue](https://github.com/shfshanyue)
:::

有现成的公网 IP 服务提供，根据 TCP 连接获得真实 IP 地址

```bash
$ curl ifconfig.me
```
