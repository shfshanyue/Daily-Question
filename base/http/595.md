# http 缓存控制中 Cach-Control 为 public 与 private 有何区别

::: tip Author
回答者: [Kiera569](https://github.com/Kiera569)

cache-control:默认值是 private,
privite 响应只能为私有缓存，不能在用户间共享
public 公有缓存，可以在多用户间共享
